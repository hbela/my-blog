Doc Theme
Style Guide

A complete reference for the dark documentation theme. Copy the CSS, JS, and template to any project.

01

Overview

This theme is a self-contained design system for building beautiful documentation pages. It consists of three files:

  1. doc-theme.css — All styles. Drop-in ready, all classes prefixed with doc-.
  2. doc-theme.js — Sidebar toggle, scroll progress, lightbox, and active nav.
  3. template.html — Starter page with placeholder content for every component.
💡

All CSS classes use the doc- prefix to avoid conflicts with your existing styles.

05

Sections

Each documentation section follows this pattern:

html
<section class="doc-section" id="my-section">
  <div class="doc-section-header">
    <span class="doc-section-number">01</span>
    <h2>Section Title</h2>
  </div>
  <p>Content goes here...</p>
</section>

Sections animate in on page load with a staggered fade-up effect (up to 10 sections).

Callouts

Four callout variants for different contexts.

💡

Info: Use doc-callout--info for tips and helpful information.

✅

Success: Use doc-callout--success for confirmations and positive outcomes.

⚠️

Warning: Use doc-callout--warning for cautions and prerequisites.

🚨

Danger: Use doc-callout--danger for destructive actions and critical warnings.

Screenshot Cards

Display UI screenshots in browser-like cards with a toolbar and click-to-zoom lightbox.

html
<div class="doc-screenshot">
  <div class="doc-screenshot-toolbar">
    <span class="dot"></span>
    <span class="dot"></span>
    <span class="dot"></span>
    <span class="label">Caption</span>
  </div>
  <img src="path/to/image.png" alt="..." />
</div>

The lightbox is activated automatically by doc-theme.js when clicking any .doc-screenshot element.

14

How to Use

To use this theme in another project, follow these steps:

  1. Copy the files into your project's docs folder:
files
your-project/
  docs/
    doc-theme.css     ← styles
    doc-theme.js      ← interactivity
    template.html     ← copy & customize
    images/           ← your screenshots
  1. Copy template.html and rename it (e.g. USER_MANUAL.html).
  2. Customize the sidebar — Update the brand name, icon letter, nav groups, and links.
  3. Add your content — Use the component patterns from this style guide.
  4. Add screenshots — Place images in docs/images/ and use the doc-screenshot card pattern.
✅

That's it! No build step required. Just HTML, CSS, and JS.

Customizing Colors

Override any CSS variable in your HTML to change the accent color:

css
/* Example: Switch to emerald accent */
:root {
  --doc-accent:        #34d399;
  --doc-accent-glow:   rgba(52, 211, 153, 0.15);
  --doc-accent-bright: #6ee7b7;
  --doc-border-accent: rgba(52, 211, 153, 0.25);
}
